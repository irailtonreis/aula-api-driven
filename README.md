## Aula Api node.js
Aula imersão devs driven

## Instalação
```sh
git clone https://github.com/irailtonreis/aula-api-driven.git
```
```sh
cd  aula-api-driven
npm i
```
executará o servidor na porta:3000 
```sh
npm run dev
```
